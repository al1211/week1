Background Color Changer

A simple and interactive web application that allows users to change the background color of the page by clicking different buttons.

🚀 Features
Change background color with a single click
Multiple color options (Red, Yellow, Black, Purple, Green, Blue)
Reset to default theme
Clean and minimal UI
🛠️ Tech Stack
HTML5
CSS3

JavaScript (Vanilla JS)
📂 Project Structure
📁 bg-color-changer
 ├── index.html
 └── README.md
▶️ How to Run


Download or clone the repository
Open index.html in your browser


💡 How It Works
All buttons are selected using querySelectorAll
Event listeners are added to each button
On click, the button's class name is used as the background color
Default button resets the background to dark mode (#111)
