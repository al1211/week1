# Centered Text Layout

## 📌 Description
This project centers the text "Hello World" both vertically and horizontally on the page.

## 🚀 Features
- Horizontally centered text
- Vertically centered text
- Clean and simple layout

## 🛠️ Tech Stack
- HTML
- CSS

## ⚙️ Implementation
Used Flexbox to align the text at the center of the page.

## 💡 What I Learned
- How to center elements using Flexbox
- Difference between horizontal and vertical alignment
- Using `justify-content` and `align-items`

## ▶️ How to Run
1. Clone or download the project
2. Open `index.html` in your browser
